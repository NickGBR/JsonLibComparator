для конфигурации приложения используй property.property

external_file_path - uri внешнего файла
core_file_path - uri нашего файла
check_type - тип проверки:
                            0 - найти все версии зависимостей отличные от верии внешнего файла,
                                найти все зависимости которые есть во внешнем файле, но нет в нашем.
                            1 - исключить зависимости которые есть во внешнем файле, но нет в нашем.
                            2 - исключить зависимости которые есть во внешнем файле, но нет в нашем,
                                а такоже @babel зависимости

                                (поиск в формате "Зависимость:номер версии во внешнем файле > номер версии в нашем файле")
                            3 - найти все версии зависимостей отличные от верии внешнего файла,
                                найти все зависимости которые есть во внешнем файле, но нет в нашем.
                            4 - исключить зависимости которые есть во внешнем файле, но нет в нашем.
                            5 - исключить зависимости которые есть во внешнем файле, но нет в нашем,
                                а такоже @babel зависимости
output_file_name - имя файла для сохранения результата работы приложения.
